How the run program: <br>
	1. On a Linux PC, open folder containing executable <br>
	2. Right-click white space click on "Open Terminal Here" <br>
	3. In the Terminal window <br>
		a. compile file: 'g++ TA.cpp -lpthread -o TA' <br>
		b. run file ./TA <br>
